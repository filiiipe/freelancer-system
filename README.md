# Freelancer System
API para gerenciar freelancers e clientes, desenvolvida com Spring Boot.

## Funcionalidades
- Cadastro de freelancers.
- Gerenciamento de projetos.
- Busca de freelancers por especialidade.

## Tecnologias
- Java 21
- Spring Boot
- PostgreSQL
- JWT (futuramente).
